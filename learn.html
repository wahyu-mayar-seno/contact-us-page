<h1>

Judul Utama

Tag <h1> digunakan untuk membuat judul utama pada halaman.

<p>

Ini adalah paragraf teks.

Tag <p> digunakan untuk membuat paragraf dalam HTML.

<strong>

Text yang ditekankan

Tag <strong> digunakan untuk memberi penekanan pada teks, biasanya dengan huruf tebal.

<em>

Text yang miring

Tag <em> digunakan untuk menekankan teks dengan memberi efek miring.

<a>

Kunjungi Example

Tag <a> digunakan untuk membuat hyperlink yang mengarah ke halaman lain.

<img>

Tag <img> digunakan untuk menyisipkan gambar dalam halaman web.

<ul>

  • Item 1
  • Item 2

Tag <ul> digunakan untuk membuat daftar tidak terurut (bullet list).

<ol>

  1. Item 1
  2. Item 2

Tag <ol> digunakan untuk membuat daftar terurut dengan nomor atau huruf.

<li>

  • Item dalam daftar

Tag <li> digunakan untuk mendefinisikan item dalam daftar terurut atau tidak terurut.

<table>

Data 1 Data 2

Tag <table> digunakan untuk membuat tabel dalam halaman web.

<tr>

Row 1

Tag <tr> digunakan untuk mendefinisikan baris dalam tabel.

<th>

Judul Kolom

Tag <th> digunakan untuk mendefinisikan sel header dalam tabel.

<td>

Data

Tag <td> digunakan untuk mendefinisikan sel data dalam tabel.

<form>

Tag <form> digunakan untuk membuat formulir untuk mengumpulkan data dari pengguna.

<input>

Tag <input> digunakan untuk mendefinisikan elemen input, seperti teks, radio, checkbox, dan lainnya.

<button>

Tag <button> digunakan untuk membuat tombol interaktif di halaman.

<select>

Tag <select> digunakan untuk membuat dropdown menu.

<option>

Tag <option> digunakan untuk mendefinisikan pilihan dalam dropdown.

<textarea>

Tag <textarea> digunakan untuk membuat area input teks yang lebih besar, seperti untuk komentar.

<header>

Header Halaman

Tag <header> digunakan untuk mendefinisikan bagian atas halaman atau seksi tertentu.

<footer>

Footer Halaman

Tag <footer> digunakan untuk mendefinisikan bagian bawah halaman atau seksi tertentu.

<section>

Bagian Konten

Tag <section> digunakan untuk mendefinisikan bagian atau seksi dalam halaman.

<article>

Artikel

Tag <article> digunakan untuk mendefinisikan artikel atau konten yang dapat berdiri sendiri.

<nav>

Tautan 1

Tag <nav> digunakan untuk mendefinisikan bagian navigasi dalam halaman web.

<aside>

Konten Sampingan

Tag <aside> digunakan untuk mendefinisikan konten yang terkait tetapi tidak langsung dengan konten utama.

<figure>

Tag <figure> digunakan untuk mengelompokkan gambar atau media lainnya dengan keterangan.

<figcaption>

Tag <figcaption> digunakan untuk memberikan keterangan pada elemen <figure>.

<main>

Konten utama halaman

Tag <main> digunakan untuk mendefinisikan konten utama halaman yang berhubungan dengan topik utama.

<details>

Konten lebih lanjut

Tag <details> digunakan untuk membuat elemen yang dapat dibuka untuk menampilkan lebih banyak informasi.

<summary>

Ringkasan

Konten lebih lanjut

Tag <summary> digunakan untuk memberikan judul atau ringkasan pada elemen <details>.

<progress>

Tag <progress> digunakan untuk menampilkan indikator progres atau kemajuan.

<meter>

Tag <meter> digunakan untuk menunjukkan nilai dalam skala tertentu.

<time>
